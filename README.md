*Tetris Game in C++* 
 *Description* 
This is a C++ implementation of the classic Tetris game, developed as a Final Year Project for the Programming Fundamentals course (BSDS-B).
 *It features:* 
Shadow piece to see where the block will land
Hold functionality to swap pieces
Scoring system with levels and line clears
Particle effects for completed lines
High score tracking
Pause, help, and menu screens
Technologies Used
C++ — core programming language
SFML (Simple and Fast Multimedia Library) — for graphics and input handling
Visual Studio — development environment
 *How to Run* 
1)Clone the repository
Copy code
Bash
git clone https://github.com/MehakAbbasi/tetris-game-cpp.git
2)Open the project in Visual Studio.
3)Make sure SFML is installed and linked properly.
4)Build and run the project.
 *Controls* 
Left / Right Arrow — Move piece
Up Arrow — Rotate piece
Down Arrow — Soft drop
Space — Hard drop
C — Hold piece
P — Pause
ESC — Return to Menu
 *High Scores* 
•Saves top 10 scores in highscores.txt
•Scores persist between game sessions
